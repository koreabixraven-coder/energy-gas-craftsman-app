에너지·가스 기능사 통합 TTS 학습앱 v1.01

업로드 방법
1. 이 폴더 안의 파일 전체를 GitHub 저장소에 업로드합니다.
2. GitHub Pages: Settings → Pages → Deploy from a branch → main / root 저장.
3. 배포 후 Ctrl+F5로 강력 새로고침합니다.

v1.01 변경 내용
- 설정창 실제 작동
- TTS 속도/음성/높이 조절
- 암기완료 청취 제외 ON/OFF
- 정답/해설 자동청취 ON/OFF
- 연속재생 자동 다음문제 ON/OFF
- 오답/암기완료 초기화
- 백업 저장/복원
- 공통 뼈대 앱으로 재사용 가능한 최소 작동 기능 포함
