# energy-gas-craftsman-app

에너지관리기능사·가스기능사 통합 TTS 기출 반복학습 앱입니다.

## v1.00 골격

- 에너지관리기능사 샘플 문제 5문제
- 가스기능사 샘플 문제 5문제
- 공통 핵심 샘플 문제 5문제
- TTS 전체읽기 / 연속재생
- 암기완료 / 암기완료 청취 제외
- 오답노트 기본 저장
- 중복출제 핵심문제 메뉴
- 배관기능장 앱과 캐시명 및 localStorage 저장키 분리

## 업로드 방법

GitHub 저장소에서 `Add file → Upload files`로 이 폴더의 파일 전체를 업로드합니다.

업로드 후 GitHub Pages를 켜고, 브라우저에서 강력 새로고침 후 테스트합니다.
