# energy-gas-craftsman-app v2.04 ENERGY_GAS_HIERARCHY

에너지·가스 기능사 통합 TTS 학습앱입니다.  
배관기능장 안정본 TTS 로직을 기반으로 하며, 문제/보기/정답/해설 자동읽기 흐름은 건드리지 않았습니다.

## v2.03 변경
- 가스기능사 문제풀이 구조를 계층형 카드로 수정
- 구조: 문제풀이 → 가스기능사 → 출제예상문제 → 제1장~제8장 → 세부항목 카드
- 가스기능사 기출문제 카드 유지
- 가스기능사 CBT 실전모의고사 카드 유지
- 기존 가스 세부항목을 한 화면에 전부 나열하던 방식 제거
- 실제 문제는 사용자가 제공하는 원본 이미지 기준으로 추후 입력

## 가스기능사 계층 구조
- 출제예상문제
  - 제1장 기초열역학 및 증기
  - 제2장 가스의 상태와 성질분류
  - 제3장 고압가스의 성질
  - 제4장 가스용 고압장치
  - 제5장 액화석유가스
  - 제6장 도시가스
  - 제7장 가스미터기
  - 제8장 고압가스 관련법규 요약
- 기출문제
- CBT 실전모의고사

## 업로드 안내
GitHub `energy-gas-craftsman-app` 저장소에 압축을 풀어 전체 파일을 덮어 업로드하세요.  
업로드 후 `?v=2.03`을 붙여 확인하면 캐시 영향을 줄일 수 있습니다.


## v2.04 변경
- 에너지관리기능사도 문제풀이 → 에너지관리기능사 → 출제예상문제 → 장 → 세부카드 구조로 수정
- 기출문제, CBT 대비 적중모의고사는 에너지관리기능사 첫 단계 카드로 유지
- TTS 자동읽기, 설정, 오답/암기완료 로직은 변경하지 않음
